# Mac 显示隐藏文件 命令 comand finder
defaults write com.apple.finder AppleShowAllFiles Yes && killall Finder //显示隐藏文件


defaults write com.apple.finder AppleShowAllFiles No && killall Finder //不显示隐藏文件
